# Přehled nových vlastností

Tato sekce obsahuje popisy nových vlastností a **funkcionalit WebJET CMS srozumitelným jazykem**, bez zbytečně technických formulací. Není to technická dokumentace. Cílem je jasně vysvětlit, co jednotlivé vlastnosti přinášejí zákazníkům a proč jsou pro ně hodnotné.

## Pro koho je tato sekce

Tato sekce je určena pro každého, kdo potřebuje rychle pochopit přínosy řešení WebJET CMS:

- **obchodníky**
- **pre-sales konzultantů**
- **projektových manažerů**
- **marketingové týmy**
- **zákazníky a rozhodovatele na straně klienta**

Informace jsou formulovány tak, aby se daly **přímo použít v komunikaci se zákazníkem** — v nabídkách, prezentacích, na setkáních nebo v odpovědích na otázky typu "proč WebJET?".

## Skupiny zákazníků

Rozlišujeme tři hlavní skupiny zákazníků:

- **Státní správa** — organizace veřejného sektoru, ministerstva, úřady, samosprávy. Kladou důraz na bezpečnost, legislativní soulad, přístupnost (WCAG) a dlouhodobou podporu.
- **Komerční sektor** — firmy, korporace, e-commerce. Zajímá je rychlost nasazení, flexibilita, integrace se třetími stranami a návratnost investice.
- **Programátoři / techničtí partneři** — interní vývojáři zákazníka nebo dodavatelské firmy. Zajímá je API, rozšiřitelnost, kvalita kódu, dokumentace a možnost vlastních úprav.

> **Důležité:** Hranice mezi skupinami se stírají. Dnes i státní správa i komerční sektor požaduje možnost vlastních úprav produktu (i když si je často nakonec sami nerealizují, chtějí mít tuto možnost). Proto jsou informace o technické rozšiřitelnosti relevantní pro všechny.
