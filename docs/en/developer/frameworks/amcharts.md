# Amcharts

The [amcharts](https://www.amcharts.com/) library is used to display charts. If you do not want the Amcharts logo to be displayed in the charts, you can purchase a license and enter its value into the conf. variable `amchartLicense`. If you have a commercial license for WebJET CMS, the license is automatically entered.

## Use

You can read about initialization and usage in the [Amcharts initialization](../frameworks/charts/frontend/README.md) section.
