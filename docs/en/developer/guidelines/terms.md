## Code Nomenclature

For the code, follow the naming convention defined in [Naming cheatsheet](https://github.com/kettanaito/naming-cheatsheet). The code is written in English, including comments. Historically, it also contains Slovak names and comments, but this is no longer allowed in the new code. We recommend that you also translate the comments into English when modifying the old class.
