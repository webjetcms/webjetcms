# Survey made easy

The **Survey Easy** application is a simpler and faster form of survey. Create a survey by simply entering a question and defining possible answers.

![](inquiry-simple-vote.png)

## Tab - Settings

The tab contains basic settings:

- **Survey name (question)** - the name of the survey that will be displayed to users
- **Active** - enable or disable voting, if inactive it only shows results
- **Allow multiple answers** - if enabled, the visitor can mark multiple answers when voting
- **Survey ID** - generated immutable survey ID

![](inquiry-simple-tab-basic.png)

## Card - Possible answers

The card contains a nested table for defining and managing possible responses.
