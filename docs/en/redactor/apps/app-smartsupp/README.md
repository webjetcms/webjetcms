# Live chat (SmartsUpp)

Communicate with your customers live, directly from your website.
To use this service, you must register (for free) at www.smartsupp.com. After registration, you will receive a Smartsupp key. The service offers a free version for one operator.
We recommend placing the application in the footer of the page so that it automatically appears on every page.

## Application settings

### Basic tab:

- **Your Smartsupp Key**: Enter your Smartsupp key, for example `d505a42ae6e813515gfbaae7a0eff418b87217ad`.
