# Mirroring

The **Mirroring** section offers an overview of linked folders and pages after the [Structure Mirroring](../../apps/docmirroring/README.md) action. In addition to an overview of existing links, it also offers warnings about potentially incorrect links, as well as the option to edit them.

The section consists of 3 tabs:

- [Folders](groups.md) - overview of linked page folders
- [Pages](docs.md) - overview of linked pages
- [Configuration](settings.md) - overview of configuration variables related to mirroring
